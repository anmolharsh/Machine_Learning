*** Instructions **

To run the file --> (in the terminal)
First download and extract data file from 'give google link'
Extract the data file and the assgn2 zip file in a common folder
Run:  
	$ python main.py

Terminal:
After running the file we see:
	Accuracy after step 1
	Number of components after applying PCA
	Accuracy after step 2
	New data set after removing outliers
	Number of features removed after sequential backward selection
	Set of new features
	Accuracy after step 3



Open 'bar_graph.png' to view the plot of 'Variance_ratio' vs 'Number of Components'
